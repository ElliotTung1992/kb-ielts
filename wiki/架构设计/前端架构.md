# 前端架构

## 定位

无构建工具的轻量静态前端，直接由 Spring Boot 托管（`resources/static/`）。面向开发者自用，不追求工程化复杂度。

## 页面清单

| 文件 | 功能 |
|---|---|
| `index.html` | 首页/导航入口 |
| `study.html` | 今日学习（核心流程页）|
| `words.html` | 词汇管理 |
| `phrases.html` | 短语管理 |
| `paraphrases.html` | 同义替换 |
| `pronunciation.html` | 发音要点 |
| `grammar-points.html` | 语法点 |
| `grammar-exercises.html` | 语法练习 |
| `listening.html` | 听力练习 |
| `speaking.html` | 口语话题 |
| `speaking-materials.html` | 口语素材 |
| `reading.html` | 阅读材料 |
| `writing.html` | 写作题目 |
| `writing-submissions.html` | 写作作文 |
| `mock-tests.html` | 模拟测试 |
| `training.html` | 专项训练 |
| `topic-tags.html` | 话题标签管理 |
| `stats.html` | 学习统计 |
| `profile.html` | 学习配置 |

## JS 模块

| 文件 | 职责 |
|---|---|
| `js/api.js` | 统一 fetch 封装，处理 ApiResponse 解包 |
| `js/nav.js` | 导航栏渲染与高亮 |
| `js/links.js` | 内容关联（正向链接）渲染 |
| `js/backlinks.js` | 反向链接渲染 |

## CSS

- `css/ielts.css` — 全局样式，基于 Bootstrap 扩展
- Bootstrap 通过 CDN 引入（无本地 npm）
