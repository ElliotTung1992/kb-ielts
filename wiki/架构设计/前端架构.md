# 前端架构

## 定位

无构建工具的轻量静态前端，直接由 Spring Boot 托管（`resources/static/`）。面向开发者自用，不追求工程化复杂度。

## study.html 练习模式（2026-05-12）

`study.html` 根据 `contentType` 决定渲染路径：

| contentType | 渲染区域 | 组件 |
|---|---|---|
| WORD / PHRASE | `#wordExerciseArea` | 主动打字练习（Variant A 卡片） |
| 其他 | `#flipScene` | 翻牌模式 |

**练习区 DOM 结构**：
- `.ex-prompt-card`（紫色渐变卡片）— 中文句子 + 中文释义 + 例句进度点
- `#exerciseInput`（文本框）— 用户输入
- `#exerciseHints`（内联提示词）— 逐词揭示
- `#exerciseAnswer`（绿色卡片）— 参考句 + 英文单词 + 音标
- `#ratingArea`（复用）— 最后一句答案揭示后显示

**JS 状态变量**（模块级）：`exExamples / exCursor / exHintIdx / exWords / exWord / exPhonetic / exDef / EXERCISE_TYPES`
