# 数据模型

## 实体一览（全量，含业务功能表）

### 内容表（10类）
| 表名 | 说明 |
|---|---|
| `ielts_words` | 雅思词汇，含音标、释义、词频、词表来源 |
| `ielts_phrases` | 短语/搭配，含类型（signal-word/connector/…）|
| `ielts_paraphrase_groups` | 同义替换组，含核心表达和全部同义词 |
| `ielts_pronunciation_points` | 发音要点（stress/linking/weak-form/…）|
| `ielts_grammar_points` | 语法点，含中英文讲解和常见错误 |
| `ielts_grammar_exercises` | 语法练习题（关联 grammar_point，可为 null 综合练习）|
| `ielts_listening_items` | 听力练习，含 Section 编号、题型、场景描述 |
| `ielts_speaking_topics` | 口语话题，含 Part 1/2/3 |
| `ielts_reading_items` | 阅读材料，含 ACADEMIC/GENERAL 类型 |
| `ielts_writing_tasks` | 写作题目，含 Task 1/2、范文、评分要点 |

### 辅助内容表
| 表名 | 说明 |
|---|---|
| `ielts_examples` | 统一例句表（多态关联五类内容，`content_type + content_id`）|
| `ielts_speaking_materials` | 口语素材（关联 speaking_topic）|
| `ielts_writing_submissions` | 写作作文提交记录 |
| `ielts_topic_tags` | 话题标签字典（`009-create-topic-tags.sql`）|
| `ielts_content_links` | 技能↔跨技能多态多对多关联 |

### 学习状态表
| 表名 | 说明 |
|---|---|
| `ielts_study_records` | SM-2 状态（每条内容唯一，`UNIQUE(content_type,content_id)`）|
| `ielts_review_logs` | 复习历史（含 before/after 快照字段）|
| `ielts_daily_plans` | 每日计划头（按日期唯一）|
| `ielts_daily_plan_items` | 计划条目明细（2026-05-07 新增，固化每日条目）|
| `ielts_mistake_logs` | 错题记录（AGAIN/HARD 时写入）|

### 业务功能表
| 表名 | 说明 |
|---|---|
| `ielts_study_profile` | 备考档案（目标分、考试日期、每日时长）|
| `ielts_mock_tests` | 模拟测试总分记录 |
| `ielts_mock_test_sections` | 模拟测试分项成绩（听说读写）|
| `ielts_content_quality` | 内容质量状态（DRAFT/VERIFIED/NEEDS_REVIEW）—— 建表但未接入计划过滤 |

## 公共字段约定

所有实体均有：
- `id UUID PRIMARY KEY DEFAULT gen_random_uuid()`
- `created_at TIMESTAMPTZ NOT NULL DEFAULT now()`
- `updated_at TIMESTAMPTZ NOT NULL DEFAULT now()`

## 内容实体公共标签字段

内容类实体（words/phrases/grammar…）均有：
- `difficulty SMALLINT` — 1（易）/ 2（中）/ 3（难）
- `skill_tags VARCHAR` — 逗号分隔：`listening,reading,speaking,writing`
- `topic_tags VARCHAR` — 话题标签，逗号分隔

## 关键关联

```
ielts_daily_plan  1──* ielts_daily_plan_items
                              │
                              └── content_type + content_id (多态关联)

ielts_review_logs ──── entity_type + entity_id (多态)
                        └── next_review_at, interval, ease_factor (SM-2)

ielts_content_links ── source_type + source_id
                     └── target_type + target_id
```
