# 系统概览

## 定位

独立的雅思学习辅助系统，提供内容管理、每日学习计划、间隔复习、学习统计。单体 Spring Boot 应用，前后端不分离（静态 HTML 直接由 Spring Boot 托管）。

## 技术栈

| 层 | 选型 | 说明 |
|---|---|---|
| 运行时 | Java 21 | Virtual Threads 就绪 |
| 框架 | Spring Boot 3.4.1 | Web / Validation / Actuator |
| 持久层 | MyBatis + PageHelper | XML Mapper，手写 SQL |
| 数据库 | PostgreSQL 16 | UUID 主键，TIMESTAMPTZ |
| 迁移 | Liquibase | SQL 格式 changeset |
| 前端 | 静态 HTML + Bootstrap | 无构建工具，原生 JS |
| 容器化 | Docker Compose | pg + app 两个服务 |

## 核心业务流

1. **内容录入** — CRUD API 管理词汇/语法/听力/口语/阅读/写作内容
2. **今日学习** — 系统生成每日计划，用户按计划学习
3. **复习触发** — 学习完成后写入 `ielts_review_logs`，SM-2 算法计算下次复习时间
4. **统计看板** — Dashboard 汇总今日进度、待复习数、错题率
