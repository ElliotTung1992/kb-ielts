# 词汇系统

涵盖三类词汇内容：单词（Words）、短语（Phrases）、同义替换组（Paraphrase Groups）。

## 实体

### IeltsWord
核心字段：`word`, `phonetic_uk`, `phonetic_us`, `part_of_speech`, `definition_zh`, `definition_en`, `example_sentence`, `frequency_level(1-5)`, `word_list`, `difficulty(1-3)`

### IeltsPhrase
核心字段：`phrase`, `meaning_zh`, `usage_note`, `example_sentence`, `category`

### IeltsParaphraseGroup
核心字段：`group_name`, `core_expression`, `synonyms(JSON或逗号分隔)`, `example_original`, `example_paraphrased`

## 公共标签

三者均有 `skill_tags`（听说读写）和 `topic_tags`（话题），用于过滤和今日计划生成。

## API

| 路径 | 说明 |
|---|---|
| `/api/ielts/words` | 词汇 CRUD + 分页 |
| `/api/ielts/phrases` | 短语 CRUD + 分页 |
| `/api/ielts/paraphrase-groups` | 同义替换 CRUD + 分页 |
