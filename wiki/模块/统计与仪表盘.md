# 统计与仪表盘

## Dashboard

`IeltsDashboardController` → `/api/ielts/dashboard`  
返回 `DashboardResponse`：

| 字段 | 说明 |
|---|---|
| todayCompleted | 今日已完成条目数 |
| todayTotal | 今日计划总条目数 |
| pendingReviews | 待复习数量 |
| studyStreak | 连续学习天数 |
| recentMistakes | 近期错题摘要 |

## 学习统计

`/api/ielts/study/stats` → `StudyStatsResponse`：
- 各内容类型学习量分布
- 近 N 天学习趋势
- 各技能错题率

## 模拟测试趋势

`/api/ielts/mock-tests/trend` → `MockTestTrendResponse`：
- 按时间顺序的分项分数折线图数据
- 支持前端渲染进步曲线

## 页面

- `stats.html` — 综合统计
- `mock-tests.html` — 模拟测试记录与趋势图
