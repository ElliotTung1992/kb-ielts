## 已完成实现记录（截至 2026-05-08）

### 13.1 第 1 周跑稳基线 ✅
- README 补齐启动路径
- `.env.example` + `.gitignore` 整理
- `application.yml` 支持 `spring.config.import: optional:file:.env[.properties]`
- `StartStudyRequest` 增加 `contentType` 白名单校验
- 新增 `SpacedRepetitionCalculatorTest`、`IeltsStudyControllerValidationTest`
- 8 个测试全部通过

### 13.2 输入校验与异常 ✅
- `ApiResponse` 增加 `code`、`details`、`traceId`（兼容旧 `success/message`）
- `IeltsExceptionHandler` 统一处理参数校验/类型转换/业务异常
- 十类内容列表接口增加参数边界（difficulty 1-3、page ≥ 1、size 1-100、studyStatus 枚举等）

### 13.3 今日计划固化 ✅（重要架构变更）
- 新增 `ielts_daily_plan_items` 表：固化每日计划条目
- 今日计划首次生成时写入明细；当天再次请求直接读取，**不再漂移**
- 完成进度改为按 plan item `COMPLETED` 状态统计，**不再依赖 review log 总数**
- 新学条目开始学习后回填明细的 `record_id`

### 13.4 复习算法升级 ✅（重要算法变更）
- `SpacedRepetitionCalculator` 升级为 **Spring Bean**（构造注入 `Clock`）
- 保留 `apply()` 静态方法兼容旧调用
- `ClockConfig` 统一提供 `Clock.systemDefaultZone()`
- **非词汇类内容复习规则改为阶段式**（不再一次 GOOD 直接 MASTERED）：
  - AGAIN → LEARNING, 1 天
  - HARD → LEARNING, 1-3 天
  - GOOD → 3/7/14/30+ 天推进，`rep≥4 && interval≥30` → MASTERED
  - EASY → 7/21/45+ 天推进，`rep≥3 && interval≥30` → MASTERED
- `ielts_review_logs` 增加复习前后快照字段：`before/after_status`、`before/after_interval_days`、`before/after_repetition_count`、`before/after_ease_factor`

### 13.5 统计扩展 ✅
- `StudyStatsResponse` 增加 `dueToday`、`weeklyReviews`、`monthlyReviews`、`masteryRate`

### 13.6 Liquibase 主变更集整理 ✅
- 001-008 合并为一个 `001-kb-ielts-schema.sql`
- 后续新增走独立 changelog 文件

### 13.7-13.12 前端与业务功能 ✅
- 口语素材列表 PostgreSQL 空参数修复
- 四大技能页面交互统一优化（状态筛选、缓存编辑、防重复提交）
- 关联面板支持「新建并关联」（搜索无结果时可直接创建单词/短语）
- 弹窗表单单列布局统一
- 话题标签字典（`ielts_topic_tags` + `/api/ielts/topic-tags` + `topic-tags.html`）

### 业务功能（一批实现）✅
新增表和接口：
- 备考档案：`IeltsStudyProfile` + `/api/ielts/profile`
- 错因管理：`ielts_mistake_logs` + `/api/ielts/mistakes`
- 模考记录：`ielts_mock_tests` + `ielts_mock_test_sections`
- 写作提交：`ielts_writing_submissions`
- 口语素材：`ielts_speaking_materials`
